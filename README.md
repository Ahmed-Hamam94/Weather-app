# Weather-app

## =>Description:
Weather app enable user to get regarding weather conditions of his location or the location he selected by City name.
 
## =>Architecture Pattern :
MVC.
 
## =>Technologies:
Openweathermap Api.
Networking using URLSession.
ActivityIndicator.
CoreLocation.
Delegation.
singleton.
